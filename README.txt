How to run with MAMP:
1. Install and run MAMP (free version is enough). https://www.mamp.info/en/downloads/
2. Start servers (both Apache and MySQL).

Importing the database:
1. Click "Open start page".
2. Click phpMyAdmin.
3. Add a new database by pressing New. Name it "reserve". Use default settings.
4. Go into the new database and press the "Import" tab.
5. Import the reserve.sql file and press Go.

Setting file root:
1. Open preferences and go to the "Web Server" tab.
2. Select the root folder of the project. This folder should contain index.php.

Accessing web site: http://localhost:80/index.php
